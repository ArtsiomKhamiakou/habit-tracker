# Habit Tracker

Простое веб-приложение для отслеживания привычек.

## Возможности
- Создание привычек
- Просмотр списка привычек
- Отметка о выполнении
- Статистика и счётчик дней

## Технологии
- Python + Flask
- SQLite
- HTML/CSS/JavaScript

## Установка
1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите: `python run.py`
